Media Queries

Media Queries é uma técnica de programação que permite que você altere o estilo de sua página de acordo com as características do dispositivo de visualização, como o tamanho de tela, a orientação e a resolução. Isso é chamado de "layout responsivo".

"Isso é chamado de layout responsivo."

A ideia é que, com o media queries, você possa criar uma única página que se adapte automaticamente a diferentes tamanhos de tela, permitindo que a sua página seja visualizada corretamenteem dispositivos móveis, computadores, televisões, entre outros.
